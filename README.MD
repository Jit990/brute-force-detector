A premium, interactive cybersecurity dashboard designed to simulate and detect brute-force login attempts. Featuring a high-end cyberpunk aesthetic with real-time forensic logging and system lockout capabilities.

- **Matrix Digital Rain Background**: High-performance multi-colored canvas animation.
- **Dynamic Threat Level**: Real-time analysis of login failure patterns (LOW -> CRITICAL).
- **Hardened Security Mock**: Automated system lockout after 4 failed attempts.
- **Cyber-Sentinel Dashboard**: Live visual tracking of "authorized" vs "rejected" traffic.
- **Forensic Export**: Download session logs as `.log` files for forensic analysis.
- **Node.js Backend**: Ready-to-use Express server for static hosting and API expansion.
